Using Variable Fonts

Variable font formats provide a new way to select a font style—
a slider replacing a dropdown menu. But the true advantage with the format is the ability to programmatically specify a precise point in a design space.

Fonts live as data

Custom axes allow the type designer to define
a functional or semantic purpose to formal variation — which can then programmatically respond to any kind of input
